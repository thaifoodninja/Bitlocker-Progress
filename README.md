A quick script to monitor the progress of Bitlock encryption and monitor its progress. 
It has the added benefit of detecting a stall in the encryption process and performing a quick pause and resume which for some reason really helps speed up progress. 
I first decided to create this script as a result of being unable to run BIOS updates in Dell Command Update as it would not permit applying those patches while encryption was in progress.
This is mostly aimed towards techs who build computers for their users and want to have a passive monitor to run in the background while they perform other build tasks.
AI (Claude 4.6) helped in the creation of this script, I have used and troubleshot the script myself numerous times and felt it was worth sharing.
